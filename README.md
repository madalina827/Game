# Game
Clona a jocului Chrome T-Rex Runner in python dar cu mai multe moduri
Moduri:
    1. CLASSIC
        -obiectele si norii sunt inspirate din jocul original
        -dino este chiar dinozaurul din jocul de la chrome
    2. FUN MODE
        -totul ii mai colorat, original si cu multe detalii
    3. Christmas
        -combinatie intre classic si Christmas vibe
Structura cod:
    Functii:
        - incarca_imagini
            =>in functie de modul ales se incarca imaginile specifice
        - reset_game
            =>intoarce jocul la setarile initiale
        - generate_obstacle
            =>genereaza obstacole random
            - pot sa apara una sau maxim doua obstacole in aceelasi timp
            - cactusi daca apar doi is unu langa altul
            - pasarile daca apar doua sunt la inaltimi diferite
    While:
        -implementarea jocului de la cum arata la cum se misca
